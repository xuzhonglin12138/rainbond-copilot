# AGENTS

This workspace contains the Rainbond Copilot agent configuration.

## Available Skills
- diagnose-service: Diagnose a Rainbond service component
- get-component-status: Get the current status of a component
- get-component-logs: Retrieve recent logs from a component
- restart-component: Restart a component (requires approval)
- scale-component-memory: Scale component memory (requires approval)
